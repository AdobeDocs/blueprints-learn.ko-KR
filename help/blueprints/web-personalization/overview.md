# 웹 및 모바일 개인화

웹 및 모바일 개인화에서는 여러 애플리케이션에서 프로필과 대상자 세분화를 사용하여 고객 경험을 개인화 및 최적화하는 방법을 다룹니다. 고객 행동, 인구 통계, 충성도 수준, 거래 내역 등을 활용하여 레이아웃, 클릭 유도, 컨텐츠 등을 개인화할 수 있습니다.
