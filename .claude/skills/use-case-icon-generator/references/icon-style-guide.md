# 사용 사례 아이콘 스타일 안내서

## 개요

사용 사례 아이콘은 사용 사례 카탈로그 테이블에서 시각적 식별자 역할을 합니다. 기본 1024x1024 해상도로 품질을 유지하면서 40px 디스플레이 폭에서 즉시 인식할 수 있어야 합니다.

## 기술 사양

| 속성 | 값 |
| --- | --- |
| 차원 | 1024 x 1024픽셀 |
| 형식 | PNG(8비트 RGB 또는 RGBA) |
| 파일 크기 | 900KB - 1.4MB 일반 |
| 표시 크기 | 카탈로그 테이블의 40px 너비 |
| 이름 지정 | `icon-{kebab-case-name}.png` |
| 저장소 경로 | `/help/blueprints/industry-use-cases/assets/use-case-icons/{industry}/` |

## 비주얼 스타일 지침

### 컴포지션
- **중앙 단일 주제** — 각 아이콘에는 사용 사례 개념을 나타내는 하나의 명확한 시각적 메타포가 포함되어야 합니다
- **가운데에 있는 컴포지션** — 주 피사체는 균형 있는 공백으로 가운데로 맞춰야 합니다.
- **텍스트 없음** — 40px에서는 텍스트를 읽을 수 없습니다. 시각적 메타데이터에 전적으로 의존합니다.
- **복잡한 장면 없음** — 다중 요소 컴포지션, 개체가 많은 배경 또는 서사 장면을 사용하지 마십시오.
- **굵은 실루엣** - 작은 실루엣으로도 아이콘 모양을 인식할 수 있어야 합니다.

### 색상 및 톤
- **깔끔하고 현대적인 팔레트** — 전문적인 기업 수준의 색상을 사용합니다.
- **업계 결합** - 동일한 업계 내의 아이콘은 함께 속해 있는 것처럼 느껴야 합니다.
- **고대비** — 주 주제가 배경에서 뚜렷하게 나타나는지 확인합니다.
- **네온 또는 너무 채도가 높은 색상을 사용하지 않음** — 팔레트를 세련되고 전문적으로 유지합니다.

### 스타일
- **현대적 및 전문가** — 깔끔한 라인, 세련된 마무리
- **일관된 렌더링** - 모든 아이콘이 동일한 디자인 시스템에서 나온 것으로 표시되어야 합니다.
- **3D나 플랫(flat)은 사용할 수 있습니다** — 그러나 업계 수직적 범위 내에서 일관성을 유지해야 합니다.
- **사실적 사진 분석 방지** — 일관성을 위해 일러스트레이션되거나 렌더링된 스타일입니다
- **브랜드 로고 또는 상표가 표시된 이미지 없음** — 일반 시각적 은유를 사용합니다.

### 소형 가독성 시험
완료하기 전에 이미지의 크기를 40픽셀로 조정합니다.
- 당신은 주요 주제를 식별할 수 있습니까?
- 전경과 배경 사이에 충분한 대비가 있습니까?
- 시각 노이즈로 바뀔 미세한 세부사항이 있습니까?
- 모양이 색깔 없이 선명하게 읽히나요(접근성의 경우)?

## 이미지 생성 프롬프트 템플릿

이 템플릿을 시작점으로 사용하여 각 사용 사례에 대한 제목과 세부 사항을 사용자 지정합니다.

```
A clean, modern icon illustration of {visual metaphor} representing {use case concept}. Professional corporate design style with bold shapes and clean lines. Single centered subject on a {solid/gradient} background. High contrast, vibrant but refined color palette. No text, no fine details, no complex backgrounds. The icon must be clearly recognizable when scaled down to 40 pixels. Square format, 1024x1024 pixels.
```

### 프롬프트 사용자 지정 팁

- **주제에 대해 자세히 표시** - &quot;두 개의 화려한 선물 상자가 들어 있는 선명한 빨간색 장바구니&quot;가 &quot;장바구니&quot;보다 좋습니다.
- **배경 처리 지정** — &quot;부드러운 파란색 그라데이션 배경에서&quot; 또는 &quot;부드러운 그림자가 있는 깨끗한 흰색 배경에서&quot;
- **조명 언급** — &quot;부드러운 스튜디오 조명&quot; 또는 &quot;부드러운 주변 조명&quot;은 일관성을 유지하는 데 도움이 됩니다.
- **스타일 수정자 추가** — &quot;미니멀리스트&quot;, &quot;기하학적&quot;, &quot;아이소메트릭&quot; 또는 &quot;플랫 디자인&quot;을 사용하여 미적 요소를 조정합니다
- **지원되는 경우 음성 프롬프트 포함** — &quot;텍스트 없음, 워터마크 없음, 테두리 없음, 사실적인 요소 없음&quot;
